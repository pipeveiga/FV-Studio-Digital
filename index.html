Detrás del estudio

Felipe Veiga, fundador de FV Studio.

Estudio programación en la Universidad Austral. FV Studio nació de una idea simple: ofrecer a negocios y empresas páginas web rápidas, claras y profesionales, con un proceso de trabajo definido y atención directa. La demanda real de empresas que necesitaban resolver su presencia online de manera ágil fue lo que dio origen al estudio.

La experiencia construyendo productos digitales en producción es la base con la que abordamos cada proyecto. Antes de FV Studio se desarrollaron Billify, un SaaS de facturación utilizado en México que opera con Stripe y Supabase, y hinch.ar, una app de carpool multiplataforma en React Native.

Preguntas frecuentes

Las consultas que recibimos antes de iniciar un proyecto.

Si tu duda no aparece acá abajo, escribinos por WhatsApp y la resolvemos directamente.

¿Cuáles son los precios de los servicios?

Cada servicio tiene un precio base claro: $220.000 una página web básica, $400.000 un sistema de turnos online y $60.000 el servicio de mantenimiento.

El precio final se ajusta según el alcance específico del proyecto y se confirma antes de comenzar, junto con los plazos y entregables.

¿Cuánto tarda en estar listo el proyecto?

Una página web básica suele estar lista entre 2 y 4 semanas. Un sistema de turnos online toma entre 3 y 4 semanas.

Los plazos exactos se definen al inicio del proyecto, junto con los hitos intermedios para que sepas en qué etapa estás en cada momento.

¿Puedo pedir cambios sobre la propuesta inicial?

Sí. Cada proyecto incluye hasta 3 rondas de correcciones para ajustar textos, imágenes y detalles visuales sobre la propuesta inicial.

Los cambios adicionales fuera de ese alcance se cotizan aparte, siempre con presupuesto claro antes de avanzar.

¿El dominio viene incluido?

El primer año de dominio está incluido en el servicio de página web básica. La renovación a partir del segundo año la gestiona el cliente directamente con el registrador.

Si ya contás con un dominio, lo integramos al nuevo sitio sin costo adicional.

¿La página viene optimizada para SEO?

Sí. Todos los sitios incluyen optimización SEO base: títulos, descripciones, estructura semántica, velocidad de carga, datos estructurados (schema) y sitemap.

Las estrategias de SEO más profundas, como creación recurrente de contenido o link building, se ofrecen como servicios complementarios.

¿La página funciona bien en celular?

Sí. Cada proyecto se diseña con enfoque mobile-first, priorizando la experiencia desde dispositivos móviles, donde se concentra la mayoría del tráfico actual.

Tengo una web vieja o desactualizada, ¿qué puedo hacer?

El servicio de mantenimiento está pensado para esos casos. Realizamos un diagnóstico inicial, identificamos qué se puede optimizar (velocidad, diseño, contenido, errores técnicos) y enviamos una propuesta puntual.

Si el sitio requiere ser rehecho, lo informamos con honestidad y te orientamos sobre el camino más conveniente.

¿Cómo es el seguimiento del proyecto?

Cada proyecto cuenta con seguimiento directo durante todo el proceso. Trabajamos en cercanía con el cliente, sin instancias innecesarias ni intermediarios, para que cada decisión y ajuste se resuelva con agilidad.

¿Cómo son las condiciones del servicio?

El proyecto se inicia con una seña y el saldo se abona contra la entrega final. Las condiciones comerciales se coordinan antes de comenzar, para que el alcance, los tiempos y los costos queden claros desde el inicio.

Se emite la documentación correspondiente según las condiciones acordadas para el proyecto.
